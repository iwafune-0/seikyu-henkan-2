2. 調査・確認した内容

📧 Supabase （データベースと認証のサービス） 招待機能の調査

Phase 7（バックエンド （サーバー側のプログラム） 実装）前に、Supabaseの招待機能が使えるか調査しました。

調査結果（2025年最新）

✅ 無料プランでも利用可能

  • supabase.auth.admin.inviteUserByEmail() が使える
  • メール送信も標準で含まれる
  • 追加料金なし

⚠️ メールレート制限 （送信数の上限）

  • 無料プラン: 2通/時間（SupabaseのSMTP （メール送信の仕組み） 使用時）
  • 本番環境ではカスタムSMTP推奨（SendGrid （メール送信サービス）、AWS SES （Amazonのメール送信サービス） 等）

結論:

  • 社内スタッフのみ（数人）なら無料プランで十分
  • Phase 7で実装予定
  • Phase 5でSupabaseプロジェクト作成時に実際にテスト
