## 1. サーバー起動

### 1.1 フロントエンド開発サーバー

```bash
# ポート5174で起動
cd frontend && npm run dev
```

**確認ポイント**:
- `localhost:5174` でアクセス可能か
- 環境変数が正しく読み込まれているか

### 1.2 バックエンドAPIサーバー

```bash
# ポート3001で起動
cd backend && npm run dev
```

**確認ポイント**:
- `localhost:3001` でAPIが応答するか
- Python環境が正しく設定されているか
- LibreOfficeがインストールされているか
